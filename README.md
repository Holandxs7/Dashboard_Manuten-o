# Dashboard_Manuten-o
Dashboard de Gestão de Manutenção desenvolvido em Power BI para monitoramento de ordens de serviço corretivas, preventivas e preditivas. O projeto permite acompanhar indicadores operacionais, custos, tempo de reparo, criticidade dos ativos e status das manutenções, apoiando a tomada de decisão e a identificação de gargalos nos processos.
